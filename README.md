# 0xlightning.github.io

This is the source code for my personal website.

## Technologies Used
- HTML5
- CSS3 (Custom Styles)
- JavaScript
- Playwright (for verification)

## Structure
- `index.html`: Home page
- `about.html`: Profile and skills
- `contact.html`: Contact information
- `writeups/`: Directory for CTF writeups
- `style.css`: Main stylesheet
- `script.js`: Site interactivity
